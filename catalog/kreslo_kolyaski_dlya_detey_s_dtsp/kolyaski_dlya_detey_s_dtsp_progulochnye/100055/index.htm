    Кресло - коляска для детей с ДЦП и инвалидов Vita Buggy 7050

    Кресло - коляска Vita Buggy 7050 предназначено для транспортировки детей с ДЦП и инвалидов при помощи сопровождающего лица.

    Тип кресла-коляски для детей с ДЦП и инвалидов Vita Buggy 7050

    • Механическое кресло - коляска;
    • Кресло - коляска - прогулочная;
    • Кресло - коляска для сопровождения детей с ДЦП и инвалидов.

    Конструкция кресла-коляски для детей с ДЦП и инвалидов Vita Buggy 7050

    • Рама кресла-коляски Vita Buggy 7050 изготовлена алюминиевых труб с антикоррозийным покрытием, устойчивым к механическим повреждениям.

    Настройка кресла-коляски для детей с ДЦП и инвалидов Vita Buggy 7050

    • Для правильной настройки и регулировки кресла-коляски для детей с ДЦП и инвалидов Vita Buggy 7050 прочитайте внимательно инструкцию по эксплуатации.
    • Один человек без специальных навыков сможет правильно настроить и отрегулировать кресло-коляску для детей с ДЦП и инвалидов Vita Buggy 7050 под нужды ребёнка, который будет им пользоваться.
    • Подголовник съемный, регулируемый по высоте и углу наклона.
    • Спинка на жестком основании, регулируемая по углу наклона до горизонтального положения, оснащена боковыми фиксаторами туловища.
    • Подлокотники съемные, регулируемые по высоте и ширине установки. 
    • Сиденье на жестком основании, регулируемое по глубине и углу наклона, оснащено мягким абдуктором.
    • Подножки съемные, откидные, регулируемые по высоте и углу наклона, оборудованы регулируемыми по высоте и углу наклона ложементами. 
    • Передние колеса с цельнолитыми шинами, регулируемые по высоте установки. Задние колеса с цельнолитыми или пневматическими шинами. 
    • Стояночные тормоза регулируемые по глубине установки.
    • Антиопрокидыватели регулируемые по высоте на роликовой опоре.

    Безопасность кресла-коляски для детей с ДЦП и инвалидов Vita Buggy 7050

    • Кресло - коляска Vita Buggy 7050 прошло все необходимые испытания по безопасности применения.
    • Кресло - коляска Vita Buggy 7050 признано годным к применению для использования в качестве средства перевозки инвалидов, пожилых или людей с ограниченными возможностями.
